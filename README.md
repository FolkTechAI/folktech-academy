# FolkTech Academy

Structured learning paths for understanding and building with AI. Two entry points: one for everyone, one for developers. All content designed to build real understanding, not buzzword familiarity.

## Tracks

### Track 0: AI for Everyone
For non-developers. No coding background assumed. Builds conceptual understanding and practical AI literacy.

| Course | Title | Status | Duration |
|--------|-------|--------|----------|
| 001 | What AI Actually Is | Not Started | 1-2 hours |
| 002 | AI Vocabulary You Need to Know | Not Started | 1 hour |
| 003 | Using AI Tools Effectively | Not Started | 2 hours |
| 004 | Local AI for Non-Developers | Not Started | 2-3 hours |

### Track 1: AI Foundations (Developer Track)
For working developers with zero AI background. Assumes coding ability.

| Course | Title | Status | Duration |

## Learning Paths

**"I just want to understand AI" (General Audience)**
001 > 002 > 003 (~4-5 hours)

**"I want to run AI on my own machine" (General Audience)**
001 > 002 > 004 (~5-6 hours)

**"I need to speak the language" (Developer, Hackathon Ready)**
101 > 103 (~5 hours)

**"I want to understand what is happening inside" (Developer)**
101 > 102 > 201 > 202 > 203 (~18 hours)

**"I want to build AI products" (Developer)**
101 > 103 > 301 > 302 > 304 (~17 hours)

**"I want the full picture" (Developer)**
All tracks in order (~55-65 hours)

## Structure

Each course folder contains:
- `README.md` - Course overview, objectives, prerequisites
- Course materials (.docx, .md, .pdf as appropriate)
- Code exercises and labs (where applicable)
- `resources/` - Companion materials, reference cards, supplementary reading
