# Course 304: Agentic AI & Tool Use

**Track:** Applied AI Development
**Duration:** 4-5 hours
**Prerequisites:** Course 301
**Status:** Not Started

## What You'll Learn

By the end of this course you will:
- Implement function calling with any major LLM API
- Build a tool-using agent that can take real actions
- Understand multi-step reasoning and planning loops
- Implement execution loops with proper error handling
- Understand MCP (Model Context Protocol) and how it extends agents
- Apply safety guardrails and limits to agentic systems
- Recognize common real-world agentic patterns

## Prerequisites

- Course 301: Prompt Engineering & System Design

## Course Content

Function calling deep dive, building tool-using agents, multi-step reasoning, planning and execution loops, MCP (Model Context Protocol), safety and guardrails, real-world agentic patterns.

## Who This Is For

Developers ready to build AI systems that can take actions in the world — not just generate text. Covers the full stack from basic function calling through production-grade agent architecture.
