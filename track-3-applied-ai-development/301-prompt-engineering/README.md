# Course 301: Prompt Engineering & System Design

**Track:** Applied AI Development
**Duration:** 3-4 hours
**Prerequisites:** Course 101
**Status:** Not Started

## What You'll Learn

By the end of this course you will:
- Design effective system prompts for production AI features
- Apply few-shot prompting and chain-of-thought techniques
- Get structured output (JSON mode) reliably from LLMs
- Understand prompt injection attacks and how to defend against them
- Design prompts suitable for sensitive domains like healthcare
- Test and iterate on prompts systematically

## Prerequisites

- Course 101: AI for Developers

## Course Content

System prompt architecture, few-shot prompting, chain-of-thought, structured output (JSON mode), prompt injection and safety, designing prompts for healthcare, testing prompts systematically.

## Who This Is For

Developers building AI-powered features who want to go beyond basic API calls and design prompts that are reliable, safe, and maintainable in production.
