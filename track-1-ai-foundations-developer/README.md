# Track 1: AI Foundations (Developer Track)

For working developers with zero AI background. Assumes you can code. Establishes vocabulary, mental models, and practical knowledge for building with AI.

## Courses

| # | Course | Prerequisites | Duration | Status |
|---|--------|--------------|----------|--------|
| 101 | AI for Developers | Must be a working developer | 2-3 hours | **Draft Complete** |
| 102 | Inside the Neural Network | 101 | 3-4 hours | **Draft Complete** |
| 103 | Hands-On Local AI Setup | 101 | 2-3 hours | Not Started |

## Design Principles

- Speaks developer-to-developer
- Connects AI concepts to patterns you already know (APIs, event handling, state management)
- Practical and product-focused, not academic
- Companion "Dev Connection" callouts bridge AI to frontend/backend concepts
