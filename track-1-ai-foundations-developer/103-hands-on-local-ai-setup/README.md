# Course 103: Hands-On Local AI Setup

**Track:** AI Foundations (Developer)
**Duration:** 2-3 hours
**Prerequisites:** Course 101
**Status:** Not Started

## What You'll Learn

By the end of this course you will:
- Install llama.cpp and run inference from the command line
- Download GGUF models from Hugging Face
- Choose the right quantization level for your hardware
- Benchmark tokens-per-second on your machine
- Get started with Ollama and understand how it differs from llama.cpp
- Call a local model from code (Python or your preferred language)
- Use MLX for accelerated inference on Apple Silicon

## Prerequisites

- Course 101: AI for Developers

## Course Content

Install llama.cpp, download GGUF models from Hugging Face, run inference locally, choose quantization levels, benchmark TPS, Ollama quick start, MLX on Apple Silicon. Student calls a local model from code by end of course.

## Who This Is For

Developers who want hands-on experience running AI models locally — no cloud required. Covers both macOS (with Apple Silicon MLX support) and general x86 setups.
