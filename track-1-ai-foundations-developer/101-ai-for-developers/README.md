# Course 101: AI for Developers

**Track:** AI Foundations (Developer)
**Duration:** 2-3 hours
**Prerequisites:** Must be a working developer
**Status:** Draft Complete

## Learning Objectives

By the end of this course you will:
- Speak the AI vocabulary confidently in technical conversations
- Understand training vs inference and what each involves computationally
- Know the open model ecosystem and major model families
- Understand tokens, parameters, temperature, and the generation loop
- Know model types: text, multimodal, base vs instruct, dense vs MoE
- Understand local AI: model formats, quantization, inference engines
- Navigate AI model licensing (Apache 2.0, MIT, Llama, GPL, CC)
- Understand function calling and agentic AI patterns
- Know the Gemma 4 model family and its capabilities
