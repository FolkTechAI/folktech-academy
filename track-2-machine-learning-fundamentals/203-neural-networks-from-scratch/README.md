# Course 203: Neural Networks from Scratch

**Track:** Machine Learning Fundamentals
**Duration:** 4-5 hours
**Prerequisites:** Course 202
**Status:** Not Started

## What You'll Learn

By the end of this course you will:
- Build a neural network in Python using only NumPy — no frameworks
- Implement a forward pass through multiple layers
- Implement backpropagation by hand
- Write a training loop that minimizes loss over epochs
- Visualize loss curves and interpret what they tell you
- Train a working classifier on a real dataset

## Prerequisites

- Course 202: Classification & Logistic Regression

## Course Content

Build a neural network from scratch in Python with no frameworks. Forward pass, backpropagation, training loop, visualizing loss curves. Train a working classifier.

## Who This Is For

Developers who want to truly understand how neural networks learn — not just use them. By building everything by hand, you'll understand what PyTorch and TensorFlow are doing under the hood when you eventually use them.
