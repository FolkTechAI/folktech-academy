# Course 201: Linear Regression & the Learning Process

**Track:** Machine Learning Fundamentals
**Duration:** 3-4 hours
**Prerequisites:** Course 102
**Status:** Not Started

## What You'll Learn

By the end of this course you will:
- Understand regression as a learning problem
- Implement cost functions and understand what they measure
- Apply gradient descent in one dimension and understand learning rate
- Identify overfitting vs underfitting and how to address them
- Split data into train/test sets correctly
- Build linear regression from scratch in Python

## Prerequisites

- Course 102: Inside the Neural Network

## Course Content

Regression, cost functions, gradient descent in one dimension, learning rate, overfitting vs underfitting, train/test split. Build linear regression from scratch in Python.

## Who This Is For

Developers and technical learners who want to understand machine learning from the ground up by building models from scratch — not just calling sklearn. This is where theory meets code.
