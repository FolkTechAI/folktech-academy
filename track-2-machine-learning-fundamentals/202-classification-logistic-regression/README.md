# Course 202: Classification & Logistic Regression

**Track:** Machine Learning Fundamentals
**Duration:** 3-4 hours
**Prerequisites:** Course 201
**Status:** Not Started

## What You'll Learn

By the end of this course you will:
- Understand binary classification as a distinct problem type from regression
- Know how the sigmoid function maps outputs to probabilities
- Understand decision boundaries and how they're learned
- Read and interpret confusion matrices, precision, and recall
- Understand why precision vs recall tradeoffs matter in real-world applications
- Build logistic regression from scratch in Python

## Prerequisites

- Course 201: Linear Regression & the Learning Process

## Course Content

Binary classification, sigmoid function, decision boundaries, confusion matrices, precision vs recall, why these matter for healthcare AI. Build logistic regression from scratch.

## Who This Is For

Developers continuing through the ML fundamentals track who want to understand how models make yes/no decisions — and what it means when they get those decisions wrong. Precision/recall tradeoffs are covered with healthcare examples.
