# Track 2: Machine Learning Fundamentals

Core ML concepts every AI developer should understand. Math-light, concept-heavy, code-included.

## Courses

| # | Course | Prerequisites | Duration | Status |
|---|--------|--------------|----------|--------|
| 201 | Linear Regression & the Learning Process | 102 | 3-4 hours | Not Started |
| 202 | Classification & Logistic Regression | 201 | 3-4 hours | Not Started |
| 203 | Neural Networks from Scratch | 202 | 4-5 hours | Not Started |
| 204 | Deep Learning Essentials | 203 | 4-5 hours | Not Started |

## Design Principles

- Every concept grounded in code you can run
- Build up from first principles
- Python-based exercises (no framework dependencies for core concepts)
- Students build working implementations, not just read about them
